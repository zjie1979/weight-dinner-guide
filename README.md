# 晚饭秤

一个可添加到 iPhone 主屏幕的纯前端 PWA：记录早上空腹体重和晚饭前体重，自动计算差值并按预设规则给出当晚饮食/运动提示。

## 使用方式

1. 早上空腹称重并输入早体重，数据会自动保存在本机。
2. 晚饭前再次称重并输入晚体重。
3. 查看自动计算的差值与当晚建议，可保存为当天记录。

## 差值区间

- `< 0 kg`：吃高蛋白
- `0–<0.6 kg`：吃正常晚饭
- `0.6–<1 kg`：少吃 / 轻量晚餐
- `1–<2 kg`：不吃晚饭
- `≥ 2 kg`：不吃晚饭 + 爬坡有氧

## 隐私与限制

- 无账号、无后端、无统计代码，数据仅保存在当前浏览器的 `localStorage` 中。
- 更换设备、清理 Safari 网站数据或更换域名时，记录不会自动迁移。
- 当前公开版使用 SVG 主屏幕图标；部分旧版 Safari 可能改用页面截图作为图标。
- 本工具执行的是用户设定的查表规则，不构成医疗或营养建议。身体不适时应及时进食，并根据个人健康状况咨询医生或营养师。
